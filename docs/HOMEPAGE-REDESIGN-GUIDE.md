# Lumina International School - Homepage Redesign Guide

## Design Analysis from Reference Image

### Color Scheme
- **Primary Dark Blue**: #1a2b4a (navy blue background)
- **Secondary Blue**: #2c5282 (medium blue sections)
- **Accent Orange/Yellow**: #f59e0b (buttons, highlights)
- **Light Blue**: #3b82f6 (info boxes)
- **White**: #ffffff (text on dark backgrounds)
- **Light Gray**: #f3f4f6 (light section backgrounds)

### Typography
- **Headings**: Bold, modern sans-serif (similar to Poppins or Montserrat)
- **Body Text**: Clean, readable sans-serif
- **Large Hero Text**: 48-60px
- **Section Headings**: 32-40px
- **Body Text**: 16-18px

### Layout Structure

#### 1. Hero Section
- Full-width background with student image
- Large heading: "Building Knowledge Step by Step"
- Subtitle text
- Prominent CTA button
- Dark overlay on image for text readability

#### 2. Welcome/About Section
- Two-column layout
- Left: Text content with bullet points
- Right: School image
- Light background color
- Orange "Learn More" button

#### 3. Quick Facts Section
- Dark blue background
- 4 statistics in a row
- Large numbers with labels
- Icons above each stat

#### 4. Academic Programs Section
- Alternating image-text layout
- White cards with rounded corners
- Professional student photos
- "Academic Curriculum" and "Co-curricular Activities" sections

#### 5. Admissions Section
- 4 colorful cards in a row
- Different colors for each: Pink, Yellow, Blue, Green
- Icons and text
- Call-to-action buttons

#### 6. Gallery/Events Section
- Image grid or slider
- Wave design element
- Group photos

#### 7. Footer
- Dark blue background
- Multiple columns
- Logos and certifications
- Social media icons
- Contact information
